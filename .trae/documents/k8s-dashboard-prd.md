# K8s集群管理平台 - 产品需求文档 (PRD)

## 1. 产品概述

一个轻量级的Kubernetes集群管理平台，提供集群资源的可视化监控、日志查看、事件追踪和YAML编辑功能。采用Go语言后端配合Vue 3前端，支持集群外独立部署，使用K8s原生认证机制，为运维人员提供高效、安全的集群管理体验。

**目标用户**: DevOps工程师、运维人员、开发人员
**核心价值**: 简化K8s集群管理流程，提供直观的资源监控和操作界面

## 2. 核心功能

### 2.1 用户角色

| 角色 | 认证方式 | 核心权限 |
|------|----------|----------|
| 集群管理员 | kubeconfig文件 | 查看所有资源、编辑YAML、执行pod/exec |
| 开发人员 | ServiceAccount Token | 查看指定命名空间资源、查看日志、执行exec |

### 2.2 功能模块

1. **仪表盘页面**: 集群概览、资源统计、健康状态
2. **资源管理页面**: 按资源类型分类管理，支持列表查看、详情查看、YAML编辑
3. **日志查看页面**: Pod日志实时流式查看，支持多容器切换
4. **事件页面**: 集群事件列表，支持过滤和搜索
5. **终端页面**: Pod exec交互式终端

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘 | 集群概览 | 显示集群节点数、命名空间数、资源总量统计 |
| 仪表盘 | 资源健康状态 | 显示Deployments、Pods、Services等资源状态分布 |
| 仪表盘 | 最近事件 | 显示最近10条Warning级别事件 |
| 资源列表 | 资源树形导航 | 按类别分组：工作负载、网络、存储、配置、安全、集群 |
| 资源列表 | 资源表格 | 显示资源名称、命名空间、状态、创建时间等 |
| 资源详情 | 资源信息卡片 | 显示资源详细信息和元数据 |
| 资源详情 | YAML编辑器 | Monaco编辑器，支持语法高亮、编辑、diff预览 |
| 日志查看 | 容器选择器 | 选择Pod中的容器 |
| 日志查看 | 日志输出区 | 实时流式日志显示，支持自动滚动、搜索 |

| 终端 | 容器终端 | xterm.js终端，支持pod/exec交互 |

## 3. 核心流程

### 3.1 用户认证流程
用户上传kubeconfig文件或配置ServiceAccount Token → 后端验证凭证 → 建立K8s API连接 → 跳转至仪表盘

### 3.2 资源编辑流程
选择资源类型 → 浏览资源列表 → 点击资源进入详情 → 编辑YAML → 预览diff → 确认应用 → 更新资源

### 3.3 日志查看流程
选择命名空间 → 选择Pod → 选择容器 → 实时显示日志流 → 可选暂停/搜索/下载

### 3.4 Pod Exec流程
选择Pod → 选择容器 → 打开终端 → 执行命令 → 查看输出

## 4. 用户界面设计

### 4.1 设计风格

- **主题**: 支持深色/浅色主题切换，默认深色主题
- **主色调**: 
  - 深色主题: 背景 #0d1117, 卡片 #161b22, 强调色 #58a6ff
  - 浅色主题: 背景 #ffffff, 卡片 #f6f8fa, 强调色 #0969da
- **状态颜色**: 成功 #3fb950, 警告 #d29922, 错误 #f85149, 信息 #58a6ff
- **字体**: 
  - 标题: JetBrains Mono (技术感)
  - 正文: Inter (可读性)
  - 代码: JetBrains Mono
- **布局**: 左侧固定导航栏 + 右侧内容区，顶部状态栏
- **按钮风格**: 圆角矩形，轻微阴影，hover时颜色加深
- **图标风格**: 线性图标，统一2px描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表盘 | 集群概览 | 卡片式布局，数字大字体显示，状态图标 |
| 仪表盘 | 资源健康状态 | 环形进度图，分类统计 |
| 仪表盘 | 最近事件 | 紧凑列表，事件类型图标 |
| 资源列表 | 资源导航 | 树形结构，可折叠分组，图标+文字 |
| 资源列表 | 资源表格 | 数据表格，状态标签，操作按钮 |
| 资源详情 | YAML编辑器 | Monaco编辑器，diff对比视图 |
| 日志查看 | 日志输出 | 终端风格，等宽字体，行号显示 |
| 事件列表 | 事件表格 | 时间线样式，类型颜色标记 |
| 终端 | 终端窗口 | xterm.js黑色终端，绿色/白色文字 |

### 4.3 响应式设计

- **桌面优先**: 主要面向桌面用户，最小支持1280px宽度
- **侧边栏**: 可折叠，小屏幕自动收起
- **表格**: 横向滚动，关键列优先显示

### 4.4 交互动效

- **页面切换**: 淡入淡出，200ms
- **列表加载**: 骨架屏占位
- **状态变化**: 颜色渐变过渡
- **按钮点击**: 缩放反馈
- **终端输出**: 逐行滚动动画

## 5. 支持的资源类型

### 5.1 工作负载
- Deployments, StatefulSets, DaemonSets, Jobs, CronJobs, Pods, ReplicaSets

### 5.2 网络
- Services, Ingresses, NetworkPolicies, Endpoints, EndpointSlices

### 5.3 存储
- PersistentVolumes, PersistentVolumeClaims, StorageClasses, VolumeAttachments

### 5.4 配置
- ConfigMaps, Secrets, ResourceQuotas, LimitRanges

### 5.5 安全
- ServiceAccounts, Roles, RoleBindings, ClusterRoles, ClusterRoleBindings

### 5.6 集群
- Nodes, Namespaces, CustomResourceDefinitions, Events

## 6. 非功能性需求

- **性能**: 资源列表加载 < 2s，日志延迟 < 500ms
- **安全**: 所有操作通过K8s RBAC验证，不存储凭证
- **可用性**: 支持K8s API高可用连接
- **兼容性**: 支持K8s 1.20+版本
